Understanding SHAP Values
Positive SHAP

A positive SHAP value means this feature pushed the model's output toward phishing. The larger the value, the stronger the push.

Negative SHAP

A negative SHAP value means this feature pushed the model's output toward legitimate. It is evidence against phishing.

Near Zero

A SHAP value near zero means this feature had little influence on the prediction for this particular URL.

Base Value

The base value is the model's average prediction across the training dataset — the starting point before any features are considered.
